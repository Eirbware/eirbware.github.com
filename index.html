Qu'est-ce que c'est ?

Eirbware est une association informatique de l'ENSEIRB-MATMECA mettant à disposition des élèves des services webs (voir http://eirbware.fr/).

Contenu des dépôts

Voici la liste des dépôts open-source proposés par Eirbware:

  • Formation : fichiers de présentation et de TDs pour la formation web
  • Core : coeur des applications et services web de Eirbware; ayant pour objectif dde centraliser le code et la base de données des utilisateurs
